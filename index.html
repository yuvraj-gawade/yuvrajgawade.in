EVO:Rack

Description:

The product deals with a software virtualization layer developed for hardware racks shipped by VMware. The product helps to manage the entire rack lifecycle including the management of Hosts, Disks, CPU, Memory, within the rack ecosystem along with providing maintenance and other various activites on the rack. The product also had a Hardware Management layer which dealt with direct communication with the physical Hosts / CPU / Memory. I was involved in the communication between the software layer and the hardware layer. The other activities that I was involved in were: Displaying the Hosts / CPU / Memory / Disks all the way to the UI using Spring Service and Controllers. Displaying stats for CPU / Memory / Disk usage as a Visual Graph on the UI in angular js. Takingnetwork information form the user on the UI and pass it to the backend. In this project I strongly involved in UI development in the Angular js.

Development Environment:

Spring, Hibernet, Angular JS 1.x, jQuery, HTML5, CSS3

Roles and Responsibilities:

  • Developed Application using Angular JS framework, in that developed many utilities, directives in angular js.
  • Making ajax call to the backed and load all rack details in the UI
